Este curso te permitirá crear sitios web completos desde cero. Al finalizar el curso el alumno podrá ver el código fuente de cualquier website y comprender completamente que hace cada línea.También podrás llevar a cabo tus ideas y crear sitios web complejos para terceros.

Maquetación HTML5

Aprenderemos conceptos básicos y avanzados sobre HTML. Como insertar imágenes, realizar hipervinculos y formularios web compatibles con smartphones y navegadores de escritorio.

Técnicas Cross-Browser

La web evoluciona a un ritmo acelerado. El código HTML es ahora más semántico y limpio. Pero los navegadores antiguos se siguen utilizando es por esto que vamos a entender las principales técnicas para que nuestros websites sean funcionales en la mayor cantidad de navegadores posibles.

Maquetación HTML4

Los newsletters se maquetan de una manera distinta a las páginas web convencionales. Deben ser compatibles con webmails y con clientes de email varios. Veremos como crear boletines de noticias y ver las principales diferencias entre las versiones cuatro y cinco de HTML.

Fundamentos de CSS

Las Hojas de Estilos en Cascadas (cascade styles sheets) nos permiten generar estructuras, botones, personalizar formularios y hasta crear animaciones. Nos aproximamos a la animación CSS y veremos como hacer que nuestra web se adapte a smartphones.

Frameworks CSS

Existen hojas de estilos que podemos incluir en nuestros proyectos. Denominados Framework CSS podemos usar 960grid o Boostrap para asegurarnos que nuestros códigos sean estandarizados y que nuestros proyectos tengan una estructura base sólida.

Frameworks Javascript

Utilizaremos jQuery para empoder efectos y animaciones simples con la certeza que estás serán 100% funcionales en todos los navegadores y veremos también cómo insertar “plugins” de otros desarrolladores a nuestros desarrollos. Breve introducción a AJAX.
