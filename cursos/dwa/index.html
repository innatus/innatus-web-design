Este nuevo curso es una expansión de DWI pero no es requisito haberlo hecho previamente. Aunque alumno deberá tener conocimientos de HTML y CSS. El obejtivo de este curso es desarrollar páginas web utilizando php, mysql, nodejs, gulp y sass.

Instalación de NodeJS

En esta etapa aprenderemos las nociones básicas de cliente servidor y luego crearemos una pequeña aplicación en NodeJS para familiarizarnos con el lenguaje y el gestor de paquetes npm.

Gulp, Bower y Git

Gulp es automatizador de tareas que nos permitirá dinamizar nuestros proyectos. Luego con la ayuda de Bower podremos instalar paquetes en nuestra web. Finalmente utilizaremos git en GitHub o BitBucket para versionar y tener una copia de nuestros proyectos.

Preprocesadores de CSS (SASS)

Agregaremos dinamismo y modularidad a nuestros CSS. Los preprocesadores nos permiten crear funciones, variables y podremos así tener un código más fácil de mantener y reciclar para futuros proyectos.

PHP y MySQL

En esta unidad comenzaremos de cero con PHP. Veremos nociones de programacion, creacion de archivos, servidor testing, y también conoceremos el gestor de paquetes Composer. Y el popular motor de bases de datos nos brindará soporte para almacenar información y realizar búsquedas en nuestras aplicaciones PHP y NodeJS.

WordPress

El popular administrador de contenidos o CMS nos permitirá tener un sitio web completo. Nuestros clientes podrán editar y modificar los contenidos del mismo. Nosotros lo instalaremos, cambiaremos la estética y veremos como crear paginas personalizadas especiales.

Prestashop

Prestashop es una solución de E-Commerce que lleva mas de trece años de desarrollo y continuas actualizaciones. Veremos como instalarlo y crear una tienda en un servidor. Personalizar la estética o instalar un nuevo tema y configurar MercadoPago para dejar la tienda lista para el cliente.

Perfil del módulo

Este curso te permitirá crear sitios web completos desde cero. Al finalizar el curso el alumno podrá ver el código fuente de cualquier website y comprender completamente que hace cada línea.También podrás llevar a cabo tus ideas y crear sitios web complejos para terceros.

Objetivos

Al finalizar el curso el alumno podrá instalar y personalizar tiendas online, sitios con wordpress ó realizalos desde cero en PHP o NodeJS. También aprenderán a automatizar y optimizar hojas de estilos, archivos javascript para sus proyectos
