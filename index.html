CSVCroquizado y Sketching Vehicular Duración: 8 clases (24hs / 3hs semanales) Presencial

Este curso te permite obtener las herramientas necesarias para el renderizado de vehículos de diferentes segmentos y utilidades. También podrás explorar diversos métodos de representación gráfica para plasmar ideas de forma efectiva.

CSPCroquizado y Sketching de Productos Duración: 8 clases (24hs / 3hs semanales) Presencial

Este curso te permite desarrollar y conocer las técnicas y los sistemas representación de dibujo a mano alzada. También te permitirá adquirir las destrezas mínimas en el uso de herramientas de renderizado.

ver +
Sketch por nuestro alumno: Francisco Guevara
DTPDocumentación Técnica de Producto Duración: 8 clases (24hs / 3hs semanales) Presencial

Este curso te permitirá modelar piezas en 3D, crear ensambles y generar los planos para documentar técnicamente un producto. El alumno se introducirá en los programas paramétricos con el concepto de diseñar un proceso y no un resultado.

ver +
Trabajo por nuestro docente: Manuel Djeredjian
MRAModelado Render y Animación Duración: 10 clases (30hs / 3hs semanales) Presencial

Este curso te permite comunicar tus ideas y diseños utilizando diferentes software para poder representarlas tridimensionalmente, asignarle apariencias y ver la forma en que interactuan los objetos en el espacio 3D.

ver +
Render por nuestro docente: Lucas Romero Di Benedetto
DWIDesarrollo Web Introductorio Duración: 10 clases (30hs / 3hs semanales) Presencial

Este curso te permitirá crear sitios web completos desde cero. Al finalizar el curso el alumno podrá ver el código fuente de cualquier website y comprender completamente que hace cada línea.También podrás llevar a cabo tus ideas y crear sitios web complejos para terceros.

ver +
Código por nuestro docente: Nicolás Danelón
DWADesarrollo Web Avanzado Duración: 8 clases (24hs / 3hs semanales) Presencial

Este nuevo curso es una expansión de DWI pero no es requisito haberlo hecho previamente. Aunque alumno deberá tener conocimientos de HTML y CSS. El obejtivo de este curso es desarrollar páginas web utilizando php, mysql, nodejs, gulp y sass.

Código por nuestro docente: Nicolás Danelón
